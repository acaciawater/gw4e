(using sqlite database)
./manage.py dumpdata --natural-foreign --natural-primary -e contenttypes -e auth.Permission > maps.json
dropdb maps
createdb maps
(switch to postgres database)
./manage.py migrate
./manage.py loaddata maps.json

when copying local psql database to server (downgrade from version 10 to version 9):
dump data only:
pg_dump -x -a -d gw4e > data.sql
init db on server with manage.py migrate
load data: 
psql -d gw4e < data.sql
